# Autism Spectrum Disorder (ASD) Prediction
## Overview
This project aims to predict Autism Spectrum Disorder (ASD) using medical, behavioral, and demographic data. By utilizing logistic regression and neural networks, the model provides accurate and reliable outcomes for ASD diagnosis.

## Key Features
### Data Sources:
Utilized a comprehensive dataset comprising medical, behavioral, and demographic data.
### Model Development:
Implemented logistic regression and neural networks (using TensorFlow and Keras) for effective prediction.
### Visualization:
Included various visualizations to represent data distribution and model performance.
